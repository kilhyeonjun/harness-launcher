# Code of conduct

## Our pledge

We want this project to be a place where people can ask questions, report bugs, and contribute without harassment or discrimination.

We will not tolerate:

- harassment, threats, or personal attacks;
- discriminatory language or behavior;
- sexualized language or unwanted attention;
- doxxing or publishing another person's private information;
- deliberately disruptive behavior in issues, pull requests, or reviews.

## Expected behavior

- Assume good intent and ask for clarification before escalating.
- Critique code and ideas, not people.
- Keep feedback specific and actionable.
- Respect project scope and maintainer time.
- Remove credentials, internal details, and personal information from public discussion.

## Enforcement

Maintainers may edit or remove comments, close or lock discussions, reject contributions, or block participants when behavior conflicts with this code of conduct.

To report a conduct concern, contact the maintainer through the [GitHub profile](https://github.com/kilhyeonjun). Do not include sensitive personal information in a public issue. Security vulnerabilities follow [SECURITY.md](SECURITY.md), not this process.

## Scope

This code applies to project spaces and to public behavior when it has a reasonable connection to the project.
